# VueXpressPro-Frontend

The initial template of the Vite+Vue project has integrated Vuex, VueRouter, Axios, and other major plug-ins.

> This project is purely used as a template without additional design.

## Technology Stack

- Vite3
- Vue3
- Vuex
- Vue Router
- Axios
- element-plus
- vueuse
- js-cookie
- normalize.css
- eslint + prettier

## Common Commands

- **Install:**
    ```bash
    npm install
    ```

- **Start up:**
    ```bash
    npm run dev
    ```

- **Compile:**
    ```bash
    npm run build
    ```

- **Format code:**
    ```bash
    npm run format
    ```

- **Production:**
    ```bash
    npm run prod
    ```
